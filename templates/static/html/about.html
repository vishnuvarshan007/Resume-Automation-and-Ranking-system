Automated Resume Processing and Ranking System

Team Members: Vishnuvarshan, Srikadir, Sudhir, Shusil Singh

Abstract

Recruitment is a time-sensitive process that requires efficient screening of resumes to identify the best candidates for a job. Traditional resume evaluation is labor-intensive and prone to inconsistencies. This project presents an Automated Resume Processing and Ranking System, a web-based platform that allows users to upload resumes, extract key information from them, and rank them based on predefined company requirements.

Introduction

This project aims to automate resume screening by extracting text from uploaded resumes and ranking them based on their relevance to job requirements, thus improving efficiency in the hiring process.

Key Features

  • Allow users to upload resumes
  • Extract text from resumes
  • Compare extracted text with job requirements
  • Assign a ranking score to each resume
  • Present a sorted list of candidates

Problem Statement

Recruiters spend significant time manually screening resumes. The process is time-consuming, error-prone, and difficult to standardize. This project aims to automate the resume screening process to make hiring more efficient.

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python (Flask)
  • Database: MySQL
  • Libraries: PyPDF2, Flask-CORS, mysql-connector-python, Flask_Mail

System Implementation

  1. Users upload resumes via a web interface.
  2. The system extracts text from PDFs using PyPDF2.
  3. Extracted text is analyzed based on job requirements.
  4. A score is assigned to each resume based on keyword presence.
  5. Resumes are ranked from best to worst fit.

Conclusion

This project automates resume screening, making hiring faster, fairer, and more accurate. Future improvements can include AI-based resume parsing and more sophisticated ranking algorithms.

References
